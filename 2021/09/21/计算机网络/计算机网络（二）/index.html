第二章 局域网技术

知识点：

  • 局域网的各种类型及其特点

  • 交换式局域网与虚拟局域网

一.局域网的各种类型及其特点

1.局域网技术的发展

token bus -> token ring -> Ethernet

  • 介质访问控制是用一条作为总线的同轴电缆连接多台计算机，对应物理层协议是 10 BASE-2 与 10 BASE-5
  • 共享介质：连接多台计算机的同轴电缆
  • 多个计算机通过一条共享介质发送和接收数据称为“多路访问”或“多路存取”

2.局域网分类

  1. token bus： 令牌控制的令牌总线形局域网
  2. token ring： 令牌控制的令牌环形局域网
  3. Ethernet： 采用带有冲突检测的载波侦听多路访问（CSMA/CD）访问控制方法的总线型Ethernet
  • 体系结构遵循：IEEE 802层次结构模型
  • 传输介质：同轴电缆、双绞线、光纤
  • 采用共享介质方式发送和接收数据帧
  • 介质访问控制都采用分布式控制方法，局域网中没有集中控制的主机

IEEE 802.1体系结构与网络互联

  • IEEE 802.3 CSMA/CD 定义CSMA/CD总线介质访问控制子层与物理层的标准
  • IEEE 802.11 无线局域网 定义无线局域网访问控制子层与物理层的标准
  • IEEE 802.15 近距离无线个人网络 定义近距离个人区域无线网络访问控制子层与物理层的标准
  • IEEE 802.16 宽带无线网络 定义宽带无线城域网访问控制子层与物理层的标准

二.Ethernet工作原理及其帧结构

1.Ethernet工作原理

CSMA/CD发送流程：先听后发，边听边发，冲突停止，延迟重发

  • CSMA/CD：随机争用型介质访问控制方法，可有效控制多主机对共享总线的访问
  • Ethernet中帧最大重发次数为16
  • 任何主机发送前都要通过CSMA/CD争取总线使用权，从发送到成功发送的等待延时不确定

2.Ethernet帧结构

  • 前导码由8B（64bit）的比特序列组成，前导码作用是实现收发双方的比特同步与帧同步，8B前导码接收后不需要保留也不计入帧头长度
  • 目的地址与源地址：帧的接收主机与发送主机的硬件地址（即物理地址即MAC地址即Ethernet地址），地址长度6B，源地址必须是48b的MAC地址，目的地址可以是单播地址、多播地址或广播地址
  • 类型字段：网络层的协议类型
  • 数据字段：网络层发送的数据部分，长度在46~1500B，加上帧头18B，Ethernet帧最大长度1518B，最小长度64B
  • 帧校验：采用32b的CRC校验，范围是：目的地址，源地址，长度，LLC数据等字段

三.交换式局域网与虚拟局域网

1.交换式局域网的基本概念

（1）共享介质 –> 交换方式 –> 交换式局域网
交换机（switch）：工作在数据链路层，根据接入交换机帧的MAC地址，过滤、转发数据帧的一种网络设备
交换机基本功能：

  • 建立和维护一个表示MAC地址与交换机端口号对应关系的映射表
  • 在发送主机与接收主机端口间建立虚连接
  • 完成帧的过滤和转发
  • 执行生成树协议，防止出现环路

（2）局域网交换机工作原理
交换机通过自我学习功能建立端口号与MAC地址对应映射表

（3）交换机的交换方式

  • 直接交换方式：交换延迟短，但缺乏错误检测能力
  • 存储转发交换方式：有帧差错检测能力，并支持不同输入速率与输出速率端口之间的帧转发，但延迟大
  • 改进直接交换方式：短Ethernet帧：直接交换；长Ethernet帧，对帧的地址字段与控制字段进行差错检测

（4）交换机的交换带宽（吞吐量）
交换带宽=端口数*端口速率（全双工模式x2）

2.虚拟局域网技术

（1）虚拟局域网的基本概念

  • 虚拟局域网建立在局域网交换机之上，以软件方式实现逻辑工作组的划分与管理
  • 虚拟局域网（VLAN）是一组逻辑上的设备和用户，这些设备和用户不受物理位置的限制

（2）VLAN划分的方法

  • 基于交换机端口的VLAN划分方法

  • 基于主机MAC地址的VLAN划分方法

  • 基于网络层地址或协议的VLAN划分方法

（3）IEEE 802.1Q 基本内容

  • 1.协议标识符：IEEE 802.1Q 增加了4B来扩展Ethernet帧。第一个字段是2B的标记协议标识符（TPID），表示该帧是IEEE802.1Q协议扩展帧，TPID取值为0x8100
  • 2.控制信息：2B的标记控制信息（TCI）第二个字段被分为3b的优先级、1b的规范格式指示符（CFI）与12b的VLAN标识符（VID）。VID长度为12b，其中0与4095被保留作科研用途，VID取值在1~4094之间

四.高速局域网-Fast、Gigabit等

1.Fast Ethernet（快速以太网）

  • fast Ethernet 传输速率达到100Mbps，但仍保留Ethernet相同帧格式与最小最大帧长度
  • IEEE 802委员会正式批准Fast Ethernet标准–IEEE 802.3u，该标准定义了介质专用接口（MII），将MAC层与物理层分开
  • 支持半双工与全双工，增加10Mbps与100Mbps速率自动协商功能

分类：

  • 100BASE-TX 两对5类非屏蔽双绞线UTP，全双工（每个主机可同时以100Mbps接收与发送数据）
  • 100BASE-T4 4对3类非屏蔽双绞线UTP，三对传输，1对冲突检测，只能半双工
  • 100BASE-FX 2芯的多模或单模光纤，全双工，用作高速主干网，从主机到集线器的多模光纤可达2km

2.Gigabit Ethernet（千兆以太网）

  • Gigabit Ethernet 传输速率达到1000Mbps，但仍保留Ethernet相同帧格式与最小最大帧长度
  • IEEE 802委员会正式批准GE标准–IEEE 802.3z，该标准定义了千兆介质专用接口（GMII），将MAC层与物理层分开

千兆介质类型：

  • 1000BASE-CX：两对屏蔽双绞线，25m
  • 1000BASE-T：4对5类非屏蔽双绞线，100m
  • 1000BASE-SX：多模光纤，550m
  • 1000BASE-LX：单模光纤，5km
  • 1000BASE-LH：单模光纤，10km
  • 1000BASE-ZX：单模光纤，70km

3.10 Gigabit Ethernet（万兆以太网）

（1）主要特点：

  • 保留Ethernet相同帧格式与最小最大帧长度
  • 万兆介质专用接口（10GMII），将MAC层与物理层分开
  • 10GbE只工作在全双工模式，传输距离只取决于光纤通信系统的性能
  • 应用领域：局域网–>城域网与广域网
  • 10GbE的物理层协议分为局域网物理层标准与广域网物理层标准

（2）物理层标准
基于光纤：（多模光纤大多<1km或800m，单模光纤传输距离远，大多数>10km）

  • 10GBASE-SR：多模光纤，300m
  • 10GBASE-LRM：多模光纤，220m
  • 10GBASE-LX4：单模光纤，10km
  • 10GBASE-LR：单模光纤，25km
  • 10GBASE-ER：单模光纤，40km
  • 10GBASE-ZR：单模光纤，80km

基于双绞线：

  • 10GBASE-CX4：6类UTP或STP双绞线，最大长度15m
  • 10GBASE-T: 6类UTP或STP双绞线，最大长度100m

实现WAN PHY标准的技术路线：
1.使用SONER/SDH光纤通道技术
2.直接采用光纤密集波分复用DWDM技术

4.100 Gigabit Ethernet

100Gigabit Ethernet 传输速率达到100Gbps的IEEE 802.3ba标准
100GbE物理接口类型：

  • 10x10GbE短距离互联的LAN接口技术
  • 4x25GE中短距离互联的LAN接口技术
  • 10m的铜缆接口和1m的系统背板互联技术

五.无线局域网技术-协议标准、802.11帧结构

1.无线局域网（WLAN）

  • 传输介质：微波、激光与红外线等无线电波，实现物理层与介质访问控制MAC子层的功能
  • 功能：
    • 传统局域网的扩充
    • 建筑物的互联
    • 移动主机漫游访问
    • 无线自组网络（Adhoc）

2.扩频无线局域网

（1）扩频通信：将信号扩展到更宽的频谱上传输

（2）跳频扩频通信：IEEE802.11规定跳频通信使用2.4GHz专用ISM频段，频率范围2.400~2.4835GHz。传输速率1Mbps或2Mbps。

（3）直接序列扩频通信（DSSS）：2.4GHz专用ISM频段，传输速率1Mbps或2Mbps。

  • 待发送数据->与伪随机数异或->调制后发送(抗干扰强)
  • 所有接收主机使用相同频段
  • 发端与收端用相同伪随机码

IEEE802.11协议规定所有无线网卡在检测到信道空闲到真正发送一帧或者发送一帧之后到发送下一帧都想要间隔一段时间：1.短帧间间隔（SIFS），2.分布协调帧间间隔（DIFS），3.点协调帧间间隔（PIFS），4.扩展帧间间隔（EIFS）

6.IEEE802.11帧结构

（1）管理帧：信标帧（无线局域网的“心跳”）、探测帧、关联帧、认证帧。
（2）控制帧：包括RTS,CTS,ACK。用于预约信道、对单播数据帧的确认。
（3）数据帧：IEEE802.11数据帧三部分组成：帧头、数据字段、帧尾。帧头30B，数据字段0~2312B，帧尾2B（校验字段）。
